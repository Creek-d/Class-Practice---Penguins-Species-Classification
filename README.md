# Penguins Species Classification
This project builds a machine learning model to classify penguin species
(Adelie, Chinstrap, Gentoo) using physical measurements.
## Features names
- culmen_length_mm
- culmen_depth_mm
- flipper_length_mm
- body_mass_g
## Labels
- Adelie → 0
- Chinstrap → 1
- Gentoo → 2
## Models
- KNN
- Decision Tree
- Random Forest
## Tech Stack
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook
